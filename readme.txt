== Description ==


Tweets Rotator 2013 :- A widget that allows you to display the tweets. Add your Twitter feed to your sidebar with this widget

Live running examples : http://themefantasy.com/blog/

Thanks to Jason Mayes who created this api. For more information visit here http://www.jasonmayes.com/projects/twitterApi/.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `tweetrotator.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the widget sections to show the twitter feed.
4 ### HOW TO CREATE A VALID ID TO USE: ###
5 Go to www.twitter.com and sign in as normal, go to your settings page.
6 Go to "Widgets" on the left hand side.
7 Create a new widget for what you need eg "user timeline" or "search" etc. 
8 Feel free to check "exclude replies" if you dont want replies in results.
9 Now go back to settings page, and then go back to widgets page, you should
10 see the widget you just created. Click edit.
11 Now look at the URL in your web browser, you will see a long number like this:
12 345735908357048478
13 Use this as your ID below instead!
